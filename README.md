Canvas Tools
============
A collection of javascript helper functions to use with (but not limited to) HTML Canvas development.

ColorUtil
---------
- hslToObject
- hslToString
- invertColor
- randomHex
- randomRGB
- rgbToObject
- rgbToString
- toGrayscale
- toHex
- toHSL
- toRGB

MathUtil
--------
- round
- sign
- constrain
- getAngle
- shortestRotation
- getDistance
- randomInt
- randomBetween
- toDegrees
- toRadians

Interactive
-----------
- getMousePosition
